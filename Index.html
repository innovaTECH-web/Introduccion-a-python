¿Qué es Python?

Python es un lenguaje de programación relativamente nuevo que fue creado por Guido van Rossum y lanzado en 1991. Su filosofía de diseño dominante es la “legibilidad”: en otras palabras, el código debe ser fácil de seguir incluso para quienes no son programadores. Utiliza mucho espacio en blanco y hace un uso eficiente de los comandos, lo que significa que es posible hacer más con menos líneas de código.

Python es también el lenguaje de programación principal que se usa con Raspberry Pi, lo que significa que puede crear una amplia gama de dispositivos interesantes con él.

Esta simplicidad y elegancia hacen de Python una gran opción para los nuevos programadores, pero también tiene mucho más que ofrecer. Para empezar, hay intérpretes disponibles en múltiples sistemas operativos, lo que significa que puede ejecutar scripts en Windows, Mac, Linux y Android. Python es también uno de los principales lenguajes de programación que se utilizan con la Raspberry Pi, lo que significa que puede crear una amplia gama de dispositivos interesantes con él y convertirlo en un lenguaje ideal para enseñar a los niños. También es ideal para el desarrollo web a través del Proyecto Django. ¡Pinterest fue escrito usando Django!

Empezando

Entonces, dicho esto, ¿cómo vamos a comenzar con Python? Si estuviera aprendiendo Python para el desarrollo de PC, comenzaría por descargar la última versión de Python 2 o Python 3 y luego un IDE (entorno de desarrollo integrado) como PyCharm. Puede obtener Python para Windows aquí.

Pero el desarrollo de PC no es lo que nos interesa aquí. Para comenzar con Python en un dispositivo Android, querrá usar QPython por ahora o QPython3. QPython es realmente un motor de script para Python 2, mientras que QPython3 ejecuta Python 3.

Python es un proyecto en curso que se mejora constantemente. Para garantizar que su código se ejecute de la manera más fluida posible, debe obtener la última versión de Python. En el momento de escribir este artículo, eso es Python 3.6.1.

La ligera complicación es que el salto de la versión 2 de Python a la versión 3 de Python fue tan significativo que rompió la compatibilidad con versiones anteriores. Eso significaba que el código escrito en Python 2 no funcionaría para Python 3 sin algunos ajustes. Eso no es un gran problema, pero lo que es un poco molesto es que algunas bibliotecas populares también se rompieron en la actualización. Una biblioteca, como sabrá, es una colección de código que otros desarrolladores pueden usar en sus propios programas y que, por lo tanto, acorta el tiempo de desarrollo y habilita una funcionalidad adicional.

Si está aprendiendo Python por primera vez, tiene sentido comenzar con Python 3 y, por lo tanto, tener el conocimiento más actualizado. Sin embargo, en el futuro, solo sepa que es posible que deba volver a Python 2 para que pueda admitir ciertas bibliotecas.

La biblioteca principal que usaremos más adelante es ‘Kivy’ y, afortunadamente, es compatible con Python 3.

Una vez que haya descargado e instalado QPython3 (que es gratuito), tendrá su propio pequeño entorno de desarrollo en el que comenzar a programar. Podrá cargar scripts desde aquí y eso será casi tan útil como crear sus propias aplicaciones nativas. Es decir que si quieres crear una herramienta básica para realizar algunas matemáticas, para ponerte a prueba en un tema, o para almacenar y recuperar datos… ¡bueno, entonces puedes hacerlo!

Y vamos a aprender a hacer ese tipo de cosas aquí mismo. Lo primero es lo primero, creemos nuestra aplicación ‘hola mundo’.

Para hacer esto, abra QPython3 y luego elija ‘Editor’. Como puede adivinar, este es el editor donde puede escribir su código o editar otros scripts. Personalmente, no puedo lidiar con este tipo de desarrollo a menos que tenga un teclado y un mouse bluetooth para trabajar, ¡pero eso es opcional!
